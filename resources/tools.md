# 既存ツール・リソース

認知的健康を維持するための既存ツールやリソースのリストです。

---

## 学習・トラッキングツール

### 学習管理

#### Notion
- **用途**: 学習ジャーナル、スキルマップの管理
- **メリット**: 柔軟なカスタマイズ
- **テンプレート例**:
  ```
  - Daily Learning Log
  - Weekly Review
  - Skill Progress Tracker
  ```
- **リンク**: [notion.so](https://notion.so)

#### Obsidian
- **用途**: 知識管理、学習ノート
- **メリット**: ローカルファースト、マークダウン、グラフビュー
- **プラグイン**:
  - Spaced Repetition
  - Daily Notes
  - Dataview（進捗追跡）
- **リンク**: [obsidian.md](https://obsidian.md)

#### Anki
- **用途**: 間隔反復学習
- **メリット**: 科学的な記憶定着
- **使い方**: 技術的な概念の復習
- **リンク**: [apps.ankiweb.net](https://apps.ankiweb.net)

## 時間管理・集中力

### ポモドーロタイマー

#### Forest
- **用途**: 集中時間の管理
- **メリット**: ゲーミフィケーション、スマホ依存防止
- **リンク**: [forestapp.cc](https://www.forestapp.cc/)

#### Toggl Track
- **用途**: 時間トラッキング
- **メリット**: 詳細な分析、プロジェクト別管理
- **活用法**: AI使用時間 vs 自力作業時間の測定
- **リンク**: [toggl.com](https://toggl.com)

#### RescueTime
- **用途**: 自動時間トラッキング
- **メリット**: アプリ/ウェブサイトごとの使用時間を自動記録
- **活用法**: ChatGPT等の使用時間を自動追跡
- **リンク**: [rescuetime.com](https://www.rescuetime.com)

## 習慣形成・目標管理

### Habitica
- **用途**: 習慣トラッキング
- **メリット**: ゲーミフィケーション、RPG風
- **活用法**:
  - Daily: 5分ルールの実践
  - Weekly: AIなしデー
  - To-Do: 月次深掘り学習
- **リンク**: [habitica.com](https://habitica.com)

### Streaks / Productive
- **用途**: 習慣の連続記録
- **メリット**: シンプル、モチベーション維持
- **活用法**: 「連続〇日学習ログ記録」の追跡

### Beeminder
- **用途**: 目標管理（金銭的コミットメント）
- **メリット**: 強制力が強い
- **活用法**: AI使用率の上限設定
- **リンク**: [beeminder.com](https://www.beeminder.com)

## ブロッキング・制限ツール

### Cold Turkey
- **用途**: アプリ/ウェブサイトブロック
- **メリット**: 強力（解除困難）、スケジュール設定
- **活用法**:
  - 「AIなしデー」にChatGPT等をブロック
  - 5分タイマー後まで使用不可
- **リンク**: [getcoldturkey.com](https://getcoldturkey.com)

### Freedom
- **用途**: マルチデバイスでのブロック
- **メリット**: スマホ・PC同時ブロック
- **リンク**: [freedom.to](https://freedom.to)

### LeechBlock (Browser Extension)
- **用途**: ブラウザレベルでのサイトブロック
- **メリット**: 無料、柔軟な設定
- **活用法**: ChatGPT/Claude等へのアクセスに制限
- **リンク**: Chrome/Firefox拡張機能

## AI活用の質を高めるツール

### Prompt管理

#### PromptBox
- **用途**: プロンプトの保存・管理
- **メリット**: 良いプロンプトを蓄積
- **活用法**: 「答えを求める質問」ではなく「思考を深める質問」のテンプレート化

#### AI Prompt Library (個人用)
- **形式**: Notion/Obsidian等でカスタム作成
- **内容**:
  ```markdown
  # 良いプロンプト例

  ## デザインレビュー
  「以下の設計について、見落としている点を指摘してください」

  ## 理解確認
  「[概念]について以下のように理解しました。正確ですか？」

  ## 仮説検証
  「[問題]の原因は[仮説]だと考えています。検証方法を提案してください」
  ```

## 統合提案：最小限のツールセット

全部使う必要はありません。以下の組み合わせで十分：

### ミニマルセット（3つ）

1. **学習ジャーナル**: Notion or Obsidian
   - 日次ログ
   - 週次レビュー
   - スキルマップ

2. **時間トラッキング**: Toggl or RescueTime
   - AI使用時間の測定
   - 作業時間の可視化

3. **習慣トラッキング**: Habitica or Streaks
   - 5分ルール
   - 週次AIなしデー
   - 月次深掘り学習

### スタンダードセット（+3つ）

4. **アルゴリズム練習**: LeetCode
   - 週次チャレンジ

5. **ドキュメント検索**: DevDocs
   - AIの前にドキュメント

6. **ブロッキング**: Cold Turkey or Freedom
   - AIなしデーの強制

### フルセット（+2つ）

7. **コミュニティ**: Discord/Slack
   - 人間からの学び

8. **ペアプロ**: Tuple or CodeTogether
   - 人間との協働

## カスタムスクリプト・ツール

### 簡易CLIツール（作成例）

#### AI使用時間トラッカー
```bash
#!/bin/bash
# ai-tracker.sh

# ChatGPT/Claude等の使用時間を記録
echo "AI session started at $(date)" >> ~/ai-usage.log

# 5分タイマー
sleep 300
osascript -e 'display notification "5 minutes passed. Did you think first?" with title "AI Usage"'
```

#### 日次リマインダー
```bash
#!/bin/bash
# daily-reminder.sh

echo "Daily Questions:"

echo "3. Do you understand everything you wrote?"
echo ""
echo "Answer (y/n) and save to journal."
```
