# 自己診断：認知的健康チェックリスト

このドキュメントは、あなたの現在の認知的健康状態とAI使用習慣を診断するためのツールです。

---

## クイック診断（5分）

以下の質問に「はい」「いいえ」で答えてください：

### 基本チェック

- [ ] **Q1**: この1週間で、AIなしで複雑な問題を解決しましたか？
- [ ] **Q2**: AIが生成したコードを、すべて理解してから使っていますか？
- [ ] **Q3**: エラーが出たとき、まず自分でエラーメッセージを読んでいますか？
- [ ] **Q4**: 新しい技術を学ぶとき、公式ドキュメントを読んでいますか？
- [ ] **Q5**: AIへの質問前に、自分で考える時間を取っていますか？
- [ ] **Q6**: この1ヶ月で、深く理解した新しい技術がありますか？
- [ ] **Q7**: 仕事で達成感を感じていますか？
- [ ] **Q8**: 技術への知的好奇心は衰えていませんか？
- [ ] **Q9**: AIの出力を批判的に評価していますか？
- [ ] **Q10**: 自分のスキルが成長していると感じますか？

### 判定

- **8-10個「はい」**: ✅ 優秀 - 健全なAI活用ができています
- **5-7個「はい」**: ⚠️ 注意 - 一部改善の余地があります
- **3-4個「はい」**: 🚨 警告 - AI依存の兆候があります
- **0-2個「はい」**: 🆘 危険 - 早急な改善が必要です

## 詳細診断（30分）

より詳細な自己分析を行います。

### Part 1: AI使用パターンの分析

#### 1.1 使用頻度

**先週のコーディング時間におけるAI使用率を推定**：

- [ ] 0-20%：ほとんど使わない
- [ ] 20-40%：時々使う
- [ ] 40-60%：半分くらい使う ← 目安の上限
- [ ] 60-80%：頻繁に使う ⚠️
- [ ] 80-100%：ほぼ常に使う 🚨

#### 1.2 使用方法

**以下のどのパターンに当てはまりますか？**（複数選択可）

**レベル1：依存的使用 🚨**
- [ ] エラーが出たら、考えずにすぐAIにコピペする
- [ ] AIの出力をそのままコピペして使う
- [ ] AIの出力を理解せずに使うことがある
- [ ] 「なぜ動くのか」より「動けばOK」と考える
- [ ] ドキュメントを読むより、AIに聞く方が早いと思う

**レベル2：受動的使用 ⚠️**
- [ ] AIに「〇〇を実装して」と丸投げする
- [ ] AIの提案を検証せずに採用することがある
- [ ] 自分で調べるより、AIに聞く方が多い
- [ ] AIがないと不安を感じる
- [ ] 難しい問題は最初からAIに頼る

**レベル3：能動的使用 ✅**
- [ ] AIに聞く前に、必ず自分で考える
- [ ] AIには「仮説の検証」を依頼する
- [ ] AIの出力を批判的に評価する
- [ ] AIを「壁打ち相手」として使う
- [ ] AIの提案を理解してから採用する

**レベル4：戦略的使用 🌟**
- [ ] AIの使用場面を意識的に選択している
- [ ] AIで時間を節約し、より難しい問題に集中
- [ ] AIから学び、自分のスキルを向上させている
- [ ] AIなしでも問題なく仕事ができる
- [ ] AIを使うことで、思考が深まっている

**判定**：
- レベル1が多い → 🚨 早急な改善が必要
- レベル2が多い → ⚠️ 注意が必要
- レベル3が多い → ✅ 良好
- レベル4が多い → 🌟 理想的

## アクションプラン

### 🌟 優秀な場合（30-40点）

**継続・強化すべきこと**：
1. 現在の良い習慣を維持
2. より難しい課題に挑戦
3. 他のエンジニアにノウハウを共有
4. 新しい技術領域に挑戦

**次のレベルへ**：
- オープンソースへの貢献
- 技術記事の執筆
- メンターとしての活動
- 研究や実験的プロジェクト

---

### ✅ 良好な場合（20-29点）

**改善すべきポイントを特定**：

診断で「⚠️」が多かった項目を確認してください。

**改善アクション**：

**学習習慣が弱い場合**
→ [weekly challenge: AIなしデー](daily-practices.md#週次チャレンジ)から開始

**AI使用が受動的な場合**
→ [Sparring Partner の使い方](daily-practices.md#2-🎯-use-ai-as-a-sparring-partner壁打ち相手として使う)を実践

**記憶・定着が弱い場合**
→ [学習ジャーナル](daily-practices.md#ツール1学習ジャーナル)を開始

**1ヶ月後に再診断してください。**

---

### ⚠️ 注意が必要な場合（10-19点）

**緊急度：中**

**即座に始めるべきこと**：

#### Week 1-2: 意識改革
1. **5分ルールの徹底**
   - AIに聞く前に必ず5分は自分で考える
   - タイマーを使って強制する

2. **日次振り返り**
   - 毎日5分、今日の学びを記録する
   - [学習ジャーナル](daily-practices.md#ツール1学習ジャーナル)を使用

#### Week 3-4: 習慣形成
3. **週次AIなしデー**
   - 週に1日、AIを使わずに作業
   - できたこと/できなかったことを記録

4. **理解ルールの徹底**
   - 理解できないコードは採用しない
   - わからない行があったら必ず調べる

#### Month 2: 深化
5. **月次深掘り学習**
   - 1つの技術を深く学ぶ
   - 公式ドキュメントを全部読む

6. **定期的な自己診断**
   - 2週間ごとにクイック診断
   - 1ヶ月後に詳細診断

**目標**：1ヶ月後に20点以上

---

### 🚨 危険な場合（0-9点）

**緊急度：高**

**このままでは**：
- 数年後、AIなしでは仕事ができなくなる
- スキルが停滞し、キャリアが行き詰まる
- 仕事の達成感が失われる
- 知的好奇心が消失する

**今すぐ始める緊急対策**：

#### 🚨 Phase 1: リセット（Week 1）

**1. AI使用の一時制限**
```
今週は以下の場合のみAIを使用可能：
- 本番障害の緊急対応
- 最終的な設計レビュー
- コードレビューの補助

それ以外は禁止。
```

**2. 基本に戻る**
```
- エラーメッセージを読む練習
- 公式ドキュメントを読む練習
- デバッガーの使い方を学ぶ
- Google検索の練習
```

**3. 小さな成功体験を積む**
```
- 簡単な問題を自力で解決
- 達成感を感じる
- 「自分でもできる」という自信を取り戻す
```

#### 🔄 Phase 2: 再構築（Week 2-4）

**4. 構造化された学習**
```
Week 2: 基礎を固める
- 使っている技術の公式ドキュメントを読む
- 基本的なコーディング問題を解く（AIなし）

Week 3: 実践
- 小さな機能を完全に自力で実装
- AIなしで1日作業してみる

Week 4: 検証
- この3週間の成長を振り返る
- 再診断を実施
```

**5. サポート体制**
```
- メンターや先輩エンジニアに相談
- 同じ課題意識を持つ仲間を見つける
- 定期的に進捗を報告する
```

#### 📈 Phase 3: 成長（Month 2-3）

**6. 健全なAI使用の再学習**
```
- Sparring Partner としての使い方を学ぶ
- 仮説検証のために使う
- 理解を深めるために使う
```

**7. 継続的な改善**
```
- 週次レビューの習慣化
- スキルマップの作成と更新
- 定期的な自己診断（2週間ごと）
```

**目標**：
- 1ヶ月後：10点以上
- 2ヶ月後：20点以上
- 3ヶ月後：25点以上

## 定期診断スケジュール

### 推奨頻度

**クイック診断**：週1回（5分）
- 毎週日曜日の夜など
- トレンドを把握

**詳細診断**：月1回（30分）
- 毎月第1日曜日など
- 深い振り返り

**総合評価**：四半期ごと（1時間）
- 3ヶ月ごと
- 長期的な変化を確認

### 記録の保持

**診断結果を記録**：
```markdown
# 診断履歴

## 2025-12-15
- クイック：7/10 ⚠️
- 詳細：18点 ⚠️
- アクション：5分ルールの徹底開始

## 2026-01-15
- クイック：8/10 ✅
- 詳細：24点 ✅
- 改善：+6点
```

**トレンド分析**：
- スコアは上昇傾向か？
- どの項目が改善したか？
- どの項目がまだ弱いか？

## 最後に：診断は手段であって目的ではない

この診断の目的は：
- ❌ 点数を競うこと
- ❌ 自分を責めること
- ❌ 完璧を目指すこと

ではなく：

- ✅ 現状を客観的に把握すること
- ✅ 改善の方向性を見つけること
- ✅ 長期的な成長を確認すること

**重要なのは、今日より明日、明日より来週、少しずつ改善していくことです。**

診断結果が悪くても、今日から始めれば必ず改善できます。

**5年後、10年後の自分のために、今日から一歩を踏み出しましょう。**
